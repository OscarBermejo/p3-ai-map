# Companies (canonical)

Approved company data lives in **`content/companies/<slug>/`** (one folder per issuer).

**Layout, scaffold, `financials/` rules, and optional `narrative/`:** [`../_meta/companies/README.md`](../_meta/companies/README.md).

**Value-chain layer** is **not** part of this path. After you add or merge a company here, record its layer in [`../_meta/company_index.yaml`](../_meta/company_index.yaml) (human-maintained).
