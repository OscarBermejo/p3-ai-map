# Intel Corporation (INTC) Financials

## Fiscal Calendar
Intel's fiscal year is a 52/53-week year ending on the last Saturday in December.
- Q1: Ends late March / early April
- Q2: Ends late June / early July
- Q3: Ends late September / early October
- Q4/Annual: Ends late December

## File Naming Convention
Quarterly files should be named `FYYYYY-Qn.yaml` (e.g., `FY2025-Q1.yaml`).
