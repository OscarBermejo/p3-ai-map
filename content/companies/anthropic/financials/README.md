# Anthropic — Financials Folder
# Private company. No SEC filings. All data from press reports, funding disclosures, and voluntary company releases.
# Follow financials_private.md sourcing rules for all files in this folder.

## Fiscal calendar
- Fiscal year: calendar year (January 1 – December 31)
- Anthropic does not report quarterly; financial data is disclosed via press reports, funding round announcements, and voluntary milestones

## Naming convention
- Annual estimates: FY{YYYY}-annual.yaml
- Where only ARR or point-in-time data is available, use period_type: annual_estimate or point_in_time per financials_private.md

## Data quality expectations
- Most standard metrics will be null (no P&L disclosure)
- Revenue and ARR are Tier 2 from The Information, Bloomberg, Reuters, CBInsights
- Funding history is Tier 1 (Form D confirmable) or strong Tier 2
- Balance sheet, gross profit, net income: not disclosed; not estimable without fabrication
